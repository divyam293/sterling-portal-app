## Implementation Details

### File Updated
- `src/lib/data/stateTaxRates.ts`

- `getStateTaxRate(state: string)`: Returns tax rate for a state
- `getStateTaxData(state: string)`: Returns full state tax data
- `calculatePremiumTax(premium: number, state: string)`: Calculates tax amount

### Usage
These rates are automatically used in:
- Admin quote entry page (`/admin/submissions/[id]/quote`)
- Tax calculation API (`/api/tax/calculate`)
- All quote calculations throughout the system

- All rates are percentages (e.g., 6 = 6%)
- Rates are applied to premium amounts
- Default rate is 2.0% if state is not found
